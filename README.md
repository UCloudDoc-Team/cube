# 概览
云主机（UHost）为您提供安全可靠的弹性计算服务。 只需几分钟，您就可以在云端获取和启用 云主机，用于实现您的计算需求。随着业务需求的变化，您可以实时扩展或缩减计算资源。云主机 支持按实际使用的资源计费，可以为您节约计算成本。使用 云主机 可以极大降低您的软硬件采购成本，简化 IT 运维工作。
